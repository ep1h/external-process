# FlatOut 2 Trainer
## Usage
 * F1 - Repair the player's vehicle.
 * F2 - Enhance acceleration performance.
 * F3 - Boost nitro efficiency.
 * F4 - Eliminate opponents.
 * F5 - Switch positions with the closest opponent.
 * LEFT/RIGHT/UP/DOWN arrows - Move opponents.
